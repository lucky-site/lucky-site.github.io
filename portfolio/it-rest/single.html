Electronic and IT systems:

Structured cabling system; In modern office buildings, structured cabling systems are implemented to support different types of local networks, including for example the telephone network or a local area network (LAN). Structured cabling system is a modern network system designed to integrate various communications equipment within a building: computer networks, telephone terminals, video surveillance systems, access control systems, door opening systems, etc.

Data storage systems such as NAS / SAN: A Network Attached Storage (NAS) is a device connected to the network functioning as a storage space available to the entire infrastructure and can be used for simple archiving or as the first point of backup and recovery following malfunctions or data loss. In practice, this is a device made up of several hard disks that are also configured to prevent failure: if one of the disks fails, the data is not lost because it is stored in such a way as to support a temporary failure of one of the installed discs. A Storage Area Network (SAN) is a data storage system connected to a network infrastructure (e.g. through a switch), usually consisting of several servers and several clients, where the server machines share the same storage space. Typically, this allows optimisation of costs related to the storage space required as well as the simplicity of a single recovery point. It allows for a quick and easy access (for data storage and recovery) to any machine connected to the network.
